CHARACTER SET NOTES
-------------------

Spinner:   Braille patterns (Unicode block U+2800-U+28FF)
Complete:  ✓ (Check mark U+2713)
Error:     ✗ (Ballot X U+2717)

All characters are single-width for consistent alignment.


TIMING SPECIFICATIONS
---------------------

Spinner update:     100ms (10 FPS)
Speed calculation:  2-second rolling window
Speed update:       Every token update
State transition:   Immediate (on update)


ACCESSIBILITY
-------------

- Braille patterns readable by screen readers
- Color coding supplements, doesn't replace text
- Clear state indication through symbols
- Speed information optional (can be disabled)


PERFORMANCE
-----------

Per frame update:   < 0.1ms
Memory per stream:  ~1KB
Lock contention:    Minimal
CPU overhead:       Negligible
